
命令行下输入
```gradle
gradle idea
```

就会生成Intellij的项目文件，真是省时省力。如果在已经存在Intellij的项目文件情况下，
想根据build.gradle中的配置来更新项目文件，可以输入
命令行下输入
```gradle
gradle cleanIdea idea
```
